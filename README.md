## About
Eat well and stay healthy! This project was developed to help people to have a healthier life, with tips and recipes to improve their health.

## Mobile
- Expo
- React Native
- Typescript
- Axios
- Expo Google Fonts
- API Clarifai

## Run the App
- /HEALTHTIPS:
> npx expo start
